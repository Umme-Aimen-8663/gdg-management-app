# GDG Team & Meeting Management App

**Flutter Fellowship 2026 – Final Project**

## Project Overview

This is a mobile application developed for Google Developer Group (GDG) chapter organizers to efficiently manage teams, schedule meetings, and track member attendance. The app supports role-based access control to ensure proper permissions for different users.

The project was built as per the requirements shared in the Flutter Fellowship 2026 group documentation.

## Features

- Role-based login (Chapter Lead, Team Lead, Member)
- Dynamic dashboard based on user role
- Team management – View and create new teams
- Meeting management – Schedule and view meetings with date, time and location
- Attendance system – Mark attendance for meetings (Present, Absent, Late, Excused)
- Clean and modern user interface

## Technologies Used

- Flutter & Dart
- Provider for state management
- Material Design 3
